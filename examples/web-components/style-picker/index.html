Style Picker Examples

The Style Picker component provides an interactive way to select colors, icons, and pictograms. This example demonstrates three variants: Single, Flat, and Disclosed.

Single Variant

The single variant displays one section at a time, ideal for focused selection of a single style element.

Flat Variant

The flat variant displays multiple sections in a single view, allowing users to see all options at once.

Disclosed Variant

The disclosed variant uses collapsible sections, perfect for organizing large sets of options into categories.
