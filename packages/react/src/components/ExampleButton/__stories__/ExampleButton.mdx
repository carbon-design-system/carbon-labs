## Getting started

Here's a quick example to get you started.

```bash
yarn add @carbon/react
yarn add @carbon-labs/react-example-button
```

### JS (via import)

```javascript
import { ExampleButton } from '@carbon-labs/react-example-button/es/index';

function App() {
  return <ExampleButton />;
}

export default App;
```

### SCSS

In your styles file import

```
@use '@carbon/react';
@use '@carbon-labs/react-example-button/scss/example-button';
```
