## Overview

Processing element is used to indicate that data is currently being
processed/loaded and the user should wait.

## Getting started

Here's a quick example to get you started.

```bash
yarn add @carbon/react
yarn add @carbon-labs/react-processing
```

### JS (via import)

```javascript
import { Processing } from '@carbon-labs/react-processing/es/index';

function App() {
  return <Processing />;
}

export default App;
```

### SCSS

In your styles file import

```
@use '@carbon/react';
@use '@carbon-labs/react-processing/scss/processing';
```
