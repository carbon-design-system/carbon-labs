## Overview

`Resizer` is an atomic component extracted from `SplitPanel` and made into an independent component. which can be reused where ever there is a resizing requirement in the ui.

<Canvas of={ResizerStories.TwoPanelsVertical} />

## Getting started

Here's a quick example to get you started.

```bash
yarn add @carbon/react
yarn add @carbon-labs/react-resizer
```

### JS (via import)

```javascript
import { Resizer } from '@carbon-labs/react-resizer/es/index';

function App() {
  return <Resizer />;
}

export default App;
```

### SCSS

In your styles file import

```
@use '@carbon/react';
@use '@carbon-labs/react-resizer/scss/resizer';
```
