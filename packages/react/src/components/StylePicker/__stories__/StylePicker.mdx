## Getting started

Here's a quick example to get you started.

```bash
yarn add @carbon/react
yarn add @carbon-labs/react-style-picker
```

### JS (via import)

```javascript
import { StylePicker } from '@carbon-labs/react-style-picker/es/index';

function App() {
  return <StylePicker />;
}

export default App;
```

### SCSS

In your styles file import

```
@use '@carbon/react';
@use '@carbon-labs/react-style-picker/scss/style-picker';
```
