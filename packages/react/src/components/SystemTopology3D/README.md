# @carbon-labs/react-system-topology-3d

3D system topology visualization built with React Three Fiber for creating
interactive data visualizations.

## Features

- 3D System Topology with React Three Fiber
- Accessibility support with keyboard navigation
- Responsive dimensions and adaptive pixel ratio
- Theme context integration
- Multiple layer system (Foundation, Core, Primary)
- Interactive controls and camera configuration
