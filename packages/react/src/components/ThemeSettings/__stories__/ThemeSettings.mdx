## Getting started

Here's a quick example to get you started.

```bash
yarn add @carbon/react
yarn add @carbon-labs/react-theme-switcher
```

### Components

- ThemeSettings: Container component for other ThemeSettings
- ThemeSwitcher: 3 way switch between light, system and dark ThemeSettings
- ThemeSetDropdown: Used if user is able to switch between theme sets
  `white/g90`, `white/g100`, `g10/g90` and `g10/g100`.
- ThemeMenuCompliment: Used if user is permitted to toggle menu compliment
  (often global menu is compliment of main theme).

### JS (via import)

```javascript
import {
  ThemeSettings,
  ThemeMenuCompliment,
  ThemeSetDropdown,
  ThemeSwitcher,
} from '@carbon-labs/react-theme-switcher/es/index';

function App() {
  const handleSwitchChange = (themeSetting) => {
    console.log(themeSetting); // 'light', 'system' or 'dark'
  };

  const handleThemeSet = (themeSet) => {
    console.log(themeSet); // value of selected themeSet
  };

  const handleMenuCompliment = (compliment) => {
    console.log(compliment); // boolean
  };

  return (
    <ThemeSettings>
      <ThemeSwitcher onChange={handleSwitchChange}></ThemeSwitcher>
      <ThemeSetDropdown
        id="theme-dropdown"
        label="Theme set"
        titleText="Theme set"
        value="g10/g100"
        onChange={handleThemeSet}
      />
      <ThemeMenuCompliment
        id="theme-menu-compliment"
        labelText="Compliment menu theme"
        checked={false}
        onChange={handleMenuCompliment}
      />
    </ThemeSettings>
  );
}

export default App;
```
