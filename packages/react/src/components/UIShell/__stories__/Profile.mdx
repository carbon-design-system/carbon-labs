## Overview

The `Profile` is a large family of compoments, composed of multiple zones; the
UserInfo, ThemeSwitcher, ReadOnlyInfo, and Links.

<Canvas of={ProfileStories.Default} />

## Profile.Root

The `Profile.Root` component is used to handle the composed components.

```javascript
<Profile.Root label="Profile" renderIcon={<UserAvatar size={20} />}>
  <Profile.UserInfo name="Thomas J. Watson" email="thomas.watson@ibm.com" />
</Profile.Root>
```

## Profile.UserInfo

The `Profile.UserInfo` goes inside the `Profile.Root` and requires a `name` prop
and an optional `email` prop. It extends IBM Product’s
[UserAvatar](https://ibm-products.carbondesignsystem.com/?path=/docs/components-useravatar--docs&globals=viewport:basic),
and supports all `UserAvatar` props. Here’s an example of how it can be used
within `<Profile.Root>`:

```javascript
<Profile.Root label="Profile" renderIcon={<UserAvatar size={20} />}>
  <Profile.UserInfo name="Thomas J. Watson" email="thomas.watson@ibm.com" />
</Profile.Root>
```

## Profile.ReadOnly

The `Profile.ReadOnly` goes inside the `Profile.Root` and accepts a list of
items that contain objects with a label and title.
