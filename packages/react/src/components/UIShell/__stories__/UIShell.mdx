## Overview

To use this package you will need to import components from both `@carbon/react`
and `@carbon-labs/react-ui-shell` to compose the UI Shell. Additional
documentation and examples for `@carbon/react` components can be found in the
[Carbon React Storybook](https://react.carbondesignsystem.com/?path=/docs/components-ui-shell-header--overview).
The following components are provided by `@carbon-labs/react-ui-shell`:

- `HeaderDivider`
- `HeaderOverflowPanel`
- `HeaderPopover`
- `HeaderPopoverActions`
- `HeaderPopoverButton`
- `HeaderPopoverContent`
- `Profile`
- `SideNav`
- `SideNavItems`
- `SideNavLink`
- `SideNavMenu`
- `SideNavMenuItem`
- `SideNavSlot`
- `SideNavTitle`
- `TrialCountdown`

## Getting started

You will need to install both `@carbon/react` and `@carbon-labs/react-ui-shell`
in your project, you will need to run the following command using
[npm](https://www.npmjs.com/):

```bash
npm install -S @carbon-labs/react-ui-shell @carbon/react
```

If you prefer [Yarn](https://yarnpkg.com/en/), use the following command
instead:

```bash
yarn add @carbon-labs/react-ui-shell @carbon/react
```

### SCSS

Import the styles from `@carbon/react` and `@carbon-labs/react-ui-shell` in your
stylesheet:

```css
@use '@carbon/react' with (
  $font-path: '@ibm/plex'
);
@use '@carbon-labs/react-ui-shell/scss/ui-shell';
```

### JS (imports)

Import components from both `@carbon/react` and `@carbon-labs/react-ui-shell`
along with any icons needed from `@carbon/icons-react` to build your UI Shell.

```javascript
import {
  SideNav,
  SideNavItems,
  SideNavMenu,
  SideNavMenuItem,
  SideNavLink,
  HeaderPopover,
  HeaderPopoverActions,
  HeaderPopoverButton,
  HeaderPopoverContent,
  HeaderContainer,
  HeaderDivider,
  HeaderOverflowPanel
  Profile
} from '@carbon-labs/react-ui-shell';
import {
  SkipToContent,
  Header,
  HeaderName,
  Theme,
  HeaderMenuButton,
  SideNavDivider,
  Content,
  Grid,
  Column,
  HeaderGlobalBar,
  HeaderGlobalAction,
  HeaderNavigation,
  HeaderMenuItem,
  HeaderMenu,
  Switcher,
  SwitcherItem,
  SwitcherDivider,
  MenuButton,
  MenuItemRadioGroup,
  ExpandableSearch,
  Link,
  Button,
} from '@carbon/react';
import { Help, Profile, Settings } from '@carbon/icons-react';
```
