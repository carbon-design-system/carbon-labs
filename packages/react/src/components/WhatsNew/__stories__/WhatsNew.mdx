## Getting started

Here's a quick example to get you started.

```bash
yarn add @carbon/react
yarn add @carbon-labs/react-whats-new
```

### JS (via import)

```javascript
import { WhatsNew } from '@carbon-labs/react-whats-new/es/index';

function App() {
  return <WhatsNew />;
}

export default App;
```

### SCSS

In your styles file import

```
@use '@carbon/react';
@use '@carbon-labs/react-whats-new/scss/whats-new';
```
