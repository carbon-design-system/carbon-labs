# Changelog

All notable changes to the "Carbon Token Snippets" extension will be documented in this file.

## [0.1.0] - 2025-01-15

### Added

- Initial release of Carbon Token Snippets extension
- SCSS snippets for Carbon Design System tokens, mixins, and functions
- Spacing tokens ($spacing-01 through $spacing-13)
- Typography style mixins
- Motion tokens and mixins
- Theme-related snippets
- Utility @use imports for Carbon modules
- BEM block class pattern generator
- Stylelint disable comments
