## 0.1.0 (2025-12-16)

### Features

* **code-snippets**: Initial release of @carbon-labs/vscode-snippets package
* Provides VSCode snippets for Carbon Design System SCSS
* Includes snippets for spacing, typography, motion, theme, and general patterns
* Automatic installation to project .vscode folder via post-install script
* Compatible with VSCode, IBM Bob, and other VSCode-based IDEs
