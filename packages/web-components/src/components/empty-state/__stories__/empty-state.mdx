## Overview

The EmptyState component follows the Carbon guidelines for empty states with
some added specifications around illustration usage.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon-labs/wc-empty-state/es/index.js';
```

<Markdown>{`${cdnJs({ components: ['empty-state'] }, packageJson)}`}</Markdown>

### HTML

The `<clabs-empty-state>` component accepts slots for `action`, `link` and
`illustration` elements, and props for `title`, `subtitle`, `size`, `kind`, and
`illustrationTheme`.

```html
<clabs-empty-state
  title="Example EmptyState title"
  subtitle="Example subtitle"
  size="lg"
  kind="notFound"
  illustrationTheme="light">
  <cds-button kind="tertiary" size="sm" slot="action">Create new</cds-button>
  <cds-link href="https://www.carbondesignsystem.com" slot="link">
    View documentation
  </cds-link>
</clabs-empty-state>
```
