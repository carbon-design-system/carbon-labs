## Overview

A rich text editor component built on [TipTap](https://tiptap.dev/) with Carbon
Design System styling. Provides text formatting, lists, tables, alignment,
colors, search, and extensible functionality through a modular extension system.

### Problem Statement

Modern web applications require rich text editing for content creation,
documentation, and collaborative editing. The Carbon Labs WYSIWYG editor
provides a fully-featured rich text editor with Carbon Design System
integration, web component architecture for framework-agnostic usage, and
TipTap's extension system for customization.

## Getting started

Here's a quick example to get you started.

### JS (via import)

```javascript
import '@carbon-labs/wc-wysiwyg/es/index.js';

// Import extensions you want to use
import {
  TextFormatting,
  Typography,
  Lists,
  History,
} from '@carbon-labs/wc-wysiwyg/es/index.js';
```

### Styles

You'll also need to import the theming tokens from `@carbon/styles` either from
npm or from our CDN helpers. Checkout our Stackblitz example above to see how
that is implemented.

<Markdown>{`${cdnJs({ components: ['wysiwyg'] }, packageJson)}`}</Markdown>
<Markdown>{`${cdnCss()}`}</Markdown>

### HTML

```html
<clabs-wysiwyg id="editor" toolbar-size="md" content="<p>Start typing...</p>">
</clabs-wysiwyg>

<script type="module">
  const editor = document.getElementById('editor');

  // Set extensions
  editor.extensions = [TextFormatting, Typography, Lists, History];

  // Listen for content changes
  editor.addEventListener('content-change', (e) => {
    console.log('HTML:', e.detail);
  });
</script>
```
